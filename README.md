# ToDoList

A clean and responsive to-do list app built with **React** and **Vite**.  
It lets you add, edit, complete, delete, search, filter, and sort tasks from a simple interface.

## Live Demo

[View the app here](https://vladsava16.github.io/ToDoList/)

## Features

- Add new tasks quickly
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by:
  - All
  - Active
  - Completed
- Sort tasks alphabetically
- Search tasks by text
- Responsive UI styled with Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS
